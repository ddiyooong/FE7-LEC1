#안드로이드 개발 #데이터 분석

💚 자기소개

안녕하세요. 사용자 중심의 문제 해결을 지향하는 개발자 신지수입니다.
안드로이드 앱 개발을 중심으로 간단한 서버 개발, 모델 연동 등 다양한 개발 경험을 쌓아왔습니다.
현재는 데이터 시각화 및 프론트엔드 개발 역량을 확장하고 있습니다.

💚 보유 기술

  • 프론트엔드: HTML, CSS, JavaScript
  • 백엔드: Java, Spring Boot, Firebase, FastAPI
  • Android: Kotlin, Jetpack Compose, Hilt
  • 데이터베이스: MySQL, Firebase Firestore
  • 기타: Git, Figma

💚 프로젝트

  1. 살래말래 (진행중) – 소비 습관 분석 및 코칭 앱
    • 소비 패턴 시각화, Firebase 기반 사용자 데이터 수집
    • OpenAI API를 활용한 가계 챗봇 코칭 기능 개발
    • 기획부터 디자인, 앱 구현까지 전반을 담당
    • 지연 응답 문제 해결을 위해 요청 흐름 구조 개선 및 Firebase 최적화
    • 기술스택: Kotlin, Jetpack Compose, Firebase, Firestore, OpenAI API
  2. 모카콩: 코딩하기 좋은 카페 – 전국 카페 추천 앱
    • 사용자 리뷰 데이터를 바탕으로 필터링 기반 추천 시스템 구현
    • 리뷰 신뢰도 확보를 위한 리뷰 수 기반 조건 도입
    • Crashlytics 로그 분석을 통해 오류 유형 분류 및 개선
    • 기술스택: Kotlin, Jetpack Compose, Hilt, MVVM, Firebase Crashlytics, Spring Boot
  3. 농장을 부탁해 – AI 기반 식물 질병 진단 앱
    • 사용자가 촬영한 사진을 FastAPI 서버로 전송해 딥러닝 모델 진단
    • UI에서 결과를 시각적으로 표현하여 사용자 이해도 향상
    • 이미지 전송 및 처리 중 오류 방지를 위한 예외처리 강화
    • 기술스택: Kotlin, FastAPI, Firebase, Python, Android
