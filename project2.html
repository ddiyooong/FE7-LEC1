농장을 부탁해: AI 기반 병충해 진단 앱

안드로이드 담당

#프로젝트 개요

“농장을 부탁해”는 스마트폰으로 작물 사진을 촬영하면 AI가 병해충 여부를 분석하고 대응법을 제공해주는 AI 기반 병충해 진단 앱입니다.
복잡한 절차 없이 누구나 손쉽게 병해충 여부를 확인할 수 있으며, 농업에 익숙하지 않은 사용자도 직관적으로 진단 결과를 이해할 수 있도록 UI/UX를 설계했습니다.

#GitHub Actions 기반 CI/CD 자동화 구축

배포 과정의 반복적인 수작업과 테스트 누락 문제를 해결하기 위해 GitHub Actions를 사용한 CI/CD 파이프라인을 구축했습니다.
main 브랜치로의 머지 시 자동 테스트, APK 빌드, 내부 테스트 트랙 업로드가 자동으로 이루어지도록 구성하여 릴리즈 과정의 안정성과 효율성을 크게 향상시켰습니다.

#카메라 UI 커스터마이징

AI 모델의 정확도는 이미지 품질에 따라 크게 좌우되므로, 기본 카메라 인텐트 대신 CameraX를 활용하여 커스텀 카메라 UI를 구현했습니다.
촬영 가이드선, 흔들림 감지, 초점 피드백 등 사용자에게 명확한 안내를 제공함으로써 진단 정확도와 사용자 만족도를 모두 향상시켰습니다.
